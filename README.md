# Coincap

Single page application for tracking cryptocurrencies and building your own portfolio.

# Main Page

-Here you see a list (table) of cryptocurrencies with basic information about them, and you can add them to your portfolio.

# Crypto page
-When a table element is clicked, a page opens with detailed information about the currency, including a field for adding it to a portfolio, as well as its history displayed as a graph.

<div>
    <img width="1420" alt="cryptoPageScreen" src="https://github.com/ReshNikita/coincap/assets/122251467/3b05a5e7-ad56-433c-81ee-344f4f53de73">

</div>

-When the "+" is clicked, a modal window opens where you can enter the quantity (including decimals) of the cryptocurrency. After submitting, the cryptocurrency is added to the portfolio in the specified quantity.

### `npm test`

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
